先别急着背语法：C 语言到底在干嘛？

学编程不是先记一堆英文单词，而是先建立一个感觉：你在写一份非常精确的“操作说明书”。

一个超级生活化的比喻

假设你让一个完全不会做饭的人煮泡面，你不能只说：“去做个泡面。”他会懵。

你需要说得非常具体：

  • 拿一个锅。
  • 倒入 500 毫升水。
  • 打开火。
  • 水开后放面饼。
  • 煮 3 分钟。
  • 倒入调料。
  • 关火。

程序也是这样。电脑很快，但它很“死板”。你必须把每一步写清楚。

核心感觉： 编程不是和电脑聊天，而是给电脑写一份严格的步骤表。你写错一步，它就照错的步骤做。

C 语言适合想明白“底层逻辑”的人

C 语言不像某些现代语言那样帮你隐藏很多细节。它会让你看到很多真实问题：

数据放在哪里？ 变量、数组、结构体，本质上都和内存有关。
程序从哪里开始？ C 程序通常从 main 函数开始执行。
为什么会崩溃？ 指针乱用、数组越界、格式写错，都可能让程序出问题。
为什么它经典？ 操作系统、嵌入式、驱动、通信协议、汽车基础软件里经常能看到 C。
完全小白可以学 C 吗？

可以，但不要用“背语法”的方式学。更好的方式是：先看懂一个小程序，再改一个小地方，再观察结果变了什么。小步快跑，比硬啃书有效得多。

1

程序是什么？代码又是什么？

先把这几个词讲清楚：程序、代码、运行、语句、注释。它们是后面所有内容的地基。

程序：一套让电脑执行的指令

程序就是一串指令。比如：

  • 显示一句话。
  • 让用户输入年龄。
  • 判断年龄是否大于 18。
  • 根据结果显示不同内容。

代码：人写给电脑看的文字

代码就是你写下来的程序说明书。C 语言代码长这样：

example.c
#include <stdio.h>

int main(void) {
    printf("我开始学 C 语言了！\n");
    return 0;
}

语句：代码里的“一句话”

在 C 语言里，很多语句要用分号 ; 结尾。你可以把分号理解成中文句号。

statement.c
printf("你好\n");   // 这是一条语句
return 0;          // 这也是一条语句

注释：写给人看的说明

注释不会被程序执行，它是给人看的备注。

comment.c
// 这是单行注释

/*
  这是多行注释
  可以写很多行
*/
本章记住： 程序是指令，代码是指令的文字形式，语句通常用分号结束，注释是写给人看的。

C 程序的基本骨架

你可以先把 C 程序看成一个固定格式的“小房子”：头文件、main 函数、代码块、返回值。

最常见骨架
#include <stdio.h>

int main(void) {
    // 你的代码写在这里
    return 0;
}

逐块解释

1
#include <stdio.h>
意思是：我要使用标准输入输出功能。比如后面要用 printf 打印内容，就经常需要它。
2
int main(void)
这是程序入口。电脑开始运行你的程序时，通常从 main 这里开始。
3
{ ... }
大括号表示一个代码块。你可以理解成“这个函数里面要做的事情”。
4
return 0;
意思是程序正常结束。这里的 0 通常代表“没出错”。
先不用完美理解： 刚开始你可以把这套骨架当模板。真正重要的是知道：你的主要代码一般写在 main 的大括号里面。

输出：让电脑说句话

学习 C 的第一个快乐瞬间，就是让电脑打印出你想要的文字。

printf 是什么？

printf 可以理解为“打印输出”。它会把内容显示到屏幕上。

hello.c
#include <stdio.h>

int main(void) {
    printf("你好，C 语言！\n");
    return 0;
}

\n 是什么？

\n 表示换行。它不会原样显示成反斜杠和 n，而是让光标跳到下一行。

没有换行 printf("A");
printf("B");
结果可能显示成：AB
有换行 printf("A\n");
printf("B\n");
结果是 A 和 B 各占一行。

打印变量：占位符

如果你要打印一个变量，不能直接把变量名写进字符串里，需要用占位符。

print_variable.c
int age = 18;
printf("我今年 %d 岁\n", age);

这里 %d 就像一个“空位”，后面的 age 会被填进去。

本章记住： printf 负责输出，\n 负责换行，%d 这种东西叫格式占位符。
4

变量：给数据起名字

变量是 C 语言里最基础的概念之一。你可以把变量理解成“贴了名字的盒子”。

为什么需要变量？

如果程序里到处写死数字，会很麻烦。比如你写一个年龄：

没有变量
printf("年龄是 18\n");

如果年龄变成 28，你就要改文字。更好的方式是用变量：

使用变量
int age = 18;
printf("年龄是 %d\n", age);

变量声明：先告诉电脑盒子类型

C 语言很严格。你要使用变量，一般要先声明它是什么类型。

变量声明
int age;       // 声明一个整数变量，名字叫 age
age = 18;       // 给 age 赋值

也可以声明和赋值一步完成：

声明并初始化
int age = 18;

变量名怎么起？

规则例子说明
可以用字母、数字、下划线student_age常见写法
不能用数字开头2age ❌应该写 age2
不要用关键字int ❌int 已经有特殊含义
尽量见名知意score ✅比 a 更容易看懂
新手大坑： 变量最好先初始化再使用。只声明不赋值，里面可能是不可预期的“垃圾值”。

数据类型：盒子也分大小和用途

变量是盒子，数据类型就是盒子的规格。整数、小数、字符，不能全都随便往一个盒子里塞。

int整数
float小数
double更精细的小数
char单个字符

最常见类型

类型适合存什么例子打印常用占位符
int整数int age = 18;%d
float小数，精度一般float price = 9.9f;%f
double小数，精度更高double pi = 3.14159;%f
char单个字符char level = 'A';%c

字符和字符串不是一回事

字符 char 用单引号：'A'
只能表示一个字符。
字符串 用双引号："Hello"
表示一串字符。
注意： 'A' 和 "A" 不一样。前者是一个字符，后者是字符串。

完整例子

types.c
#include <stdio.h>

int main(void) {
    int age = 18;
    double height = 1.75;
    char grade = 'A';

    printf("年龄：%d\n", age);
    printf("身高：%.2f\n", height);
    printf("等级：%c\n", grade);

    return 0;
}
%.2f 是什么意思？

%f 用来打印小数，.2 表示保留 2 位小数。所以 %.2f 就是“打印一个小数，保留两位”。

运算：让程序会算账

C 语言可以做加减乘除，也可以做比较、逻辑判断。先把最常见的运算学会。

算术运算符

符号含义例子结果
+加3 + 25
-减3 - 21
*乘3 * 26
/除5 / 22，整数除法会截断小数
%取余5 % 21
整数除法大坑： 5 / 2 的结果是 2，不是 2.5。因为 5 和 2 都是整数。想得到小数，可以写 5.0 / 2。

赋值运算符

= 不是“相等”的意思，而是“把右边的值放到左边变量里”。

assign.c
int age;
age = 18;   // 把 18 放进 age 里

自增和自减

i++ 表示让 i 增加 1，常用于循环。

increment.c
int i = 1;
i++;   // i 变成 2
i--;   // i 又变成 1

7

输入：让用户把数据交给程序

输出是电脑说话，输入就是人对电脑说话。C 语言里常见输入函数是 scanf。

一个输入年龄的例子

input_age.c
#include <stdio.h>

int main(void) {
    int age;

    printf("请输入你的年龄：");
    scanf("%d", &age);

    printf("你今年 %d 岁\n", age);
    return 0;
}

为什么 scanf 里要写 &age？

先用生活比喻理解：你让快递员把包裹送给你，不能只告诉他“送给张三”，最好要告诉他地址。

age 是变量名，&age 可以先理解成“age 这个盒子的地址”。scanf 需要知道把用户输入的数据放到哪里，所以要给它地址。

暂时这样记： 用 scanf 给普通变量输入时，变量名前面通常要加 &。后面学指针时会彻底解释它。

多个输入

multi_input.c
int a, b;
scanf("%d %d", &a, &b);
printf("a + b = %d\n", a + b);
新手注意： scanf 对输入格式比较敏感。刚开始先练整数、小数这类简单输入，不要一上来处理复杂文本。

判断：让程序学会“如果……就……”

程序如果只会从上到下执行，会很呆。判断语句让程序能根据不同情况做不同选择。

if / else

if_else.c
int score = 75;

if (score >= 60) {
    printf("及格\n");
} else {
    printf("不及格\n");
}

这段代码翻译成人话就是：

如果 score 大于等于 60，就打印“及格”；否则，打印“不及格”。

关系运算符

符号意思例子
>大于age > 18
<小于age < 18
>=大于等于score >= 60
<=小于等于score <= 100
==是否相等level == 1
!=是否不相等age != 0
超常见错误： = 是赋值，== 才是判断相等。if (x = 1) 和 if (x == 1) 完全不是一回事。

else if：多个分支

else_if.c
if (score >= 90) {
    printf("优秀\n");
} else if (score >= 60) {
    printf("及格\n");
} else {
    printf("不及格\n");
}

逻辑运算符：多个条件组合

符号意思例子
&&并且age >= 18 && age <= 60
||或者score < 0 || score > 100
!取反!valid

循环：重复的事情交给电脑

电脑最擅长的事之一，就是不嫌烦地重复。循环就是让一段代码反复执行。

for 循环：知道要重复几次时用

for.c
for (int i = 1; i <= 5; i++) {
    printf("第 %d 次学习 C\n", i);
}

for 的三段式

1
int i = 1
开始前做一次：创建变量 i，让它等于 1。
2
i <= 5
每一轮开始前检查：如果条件成立，就继续执行。
3
i++
每一轮结束后执行：让 i 增加 1。

while 循环：只要条件成立就继续

while.c
int count = 1;

while (count <= 5) {
    printf("count = %d\n", count);
    count++;
}
死循环： 如果循环条件永远成立，程序会一直跑下去。比如上面的 count++ 忘了写，count 就一直是 1。

break 和 continue

break 直接结束整个循环。像打游戏时“退出关卡”。
continue 跳过本轮，进入下一轮。像“这次不算，下一位”。

数组：一排同类型盒子

一个变量只能存一个值。数组可以存一组值，比如 5 个成绩、10 个温度、100 个采样点。

数组是什么？

如果变量是一个盒子，数组就是一排盒子，并且这些盒子类型相同。

scores[0]90
scores[1]85
scores[2]70
scores[3]100
array.c
int scores[5] = {90, 85, 70, 88, 100};

printf("第一个成绩：%d\n", scores[0]);
printf("第二个成绩：%d\n", scores[1]);
数组下标从 0 开始： 长度是 5 的数组，下标是 0, 1, 2, 3, 4。没有 scores[5] 这个合法位置。

遍历数组

遍历就是从头到尾访问数组里的每一个元素。

array_loop.c
int scores[5] = {90, 85, 70, 88, 100};

for (int i = 0; i < 5; i++) {
    printf("scores[%d] = %d\n", i, scores[i]);
}

数组常见用途

存成绩 计算平均分、最高分、最低分。
存传感器数据 比如一段时间内的温度或速度。
存列表 比如菜单编号、商品价格、用户编号。
11

字符串：一串字符

C 语言里没有特别高级的字符串类型。字符串本质上是一个字符数组，并且最后有一个特殊结束标记。

字符串怎么写？

string.c
char name[] = "Tom";
printf("名字：%s\n", name);

字符串背后长什么样？

"Tom" 实际上类似这样：

'T'name[0]
'o'name[1]
'm'name[2]
'\0'结束标记
'\0' 很重要： 它表示字符串到这里结束。很多字符串错误，都和忘记结束标记或数组空间不够有关。

字符数组空间要够

如果要存 "Tom"，至少需要 4 个位置：T、o、m、'\0'。

string_size.c
char name[4] = "Tom";  // 正好够
不要这样： char name[3] = "Tom"; 空间不够，因为还需要放结束标记 '\0'。
12

函数：把大问题拆成小机器

函数是写好的一段小程序。你给它输入，它帮你处理，然后可能给你一个结果。

为什么需要函数？

如果你有一段代码要用很多次，复制粘贴会很乱。函数可以把这段逻辑封装起来。

function.c
int add(int a, int b) {
    return a + b;
}

函数三件套

部分例子解释
返回类型int这个函数最后返回一个整数。
函数名add这个函数的名字。
参数int a, int b调用函数时传进来的数据。

完整使用

use_function.c
#include <stdio.h>

int add(int a, int b) {
    return a + b;
}

int main(void) {
    int result = add(3, 5);
    printf("结果：%d\n", result);
    return 0;
}

void 是什么？

void 可以表示“没有”。

int main(void) 括号里的 void 表示这个函数不接收参数。
void sayHello(void) 前面的 void 表示这个函数不返回结果。
13

作用域：变量不是到处都能用

作用域就是变量的“有效范围”。一个变量在哪个范围内声明，通常就只能在哪个范围内使用。

局部变量

写在函数里面的变量，通常叫局部变量。它只在这个函数内部有效。

scope.c
void test(void) {
    int x = 10;  // x 只在 test 函数里有效
}

代码块作用域

大括号 { } 也会形成一个范围。

block_scope.c
if (1) {
    int temp = 100;
    printf("%d\n", temp);  // 可以用
}

// printf("%d\n", temp);  // 这里通常不能再用 temp
本章记住： 变量不是声明一次就全世界都能用。先看它在哪个大括号里，再判断能不能访问。
14

指针：C 语言最吓人，但也最核心的概念

指针不用一口气吃透。先把它理解成“存地址的变量”。这已经跨过最难的第一道门了。

先理解内存地址

变量存在内存里。内存可以想象成一排排小房间，每个房间都有编号，这个编号就是地址。

10地址 1000
20地址 1004
30地址 1008
40地址 1012

&：取地址

&x 的意思是：拿到变量 x 的地址。

*：顺着地址找值

如果 p 是一个指针，*p 的意思是：顺着 p 里面存的地址，找到那个地方的值。

pointer.c
#include <stdio.h>

int main(void) {
    int x = 10;
    int *p = &x;

    printf("x 的值：%d\n", x);
    printf("通过 p 找到的值：%d\n", *p);

    return 0;
}

用快递地址理解指针

变量 像一个房间，里面放着东西。
地址 像房间门牌号。
指针 像一张写着门牌号的纸条。
指针危险在哪里？ 如果纸条上的地址是错的，你可能会去错房间，拿错东西，甚至把别人的东西改坏。C 语言不会一直拦着你，所以指针要谨慎。
为什么说指针很重要？

因为很多底层场景都需要直接或间接操作内存。数组、字符串、函数参数、动态内存、硬件寄存器、通信缓冲区，都可能和指针有关。

15

结构体：把一组相关数据打包

如果你要描述一个学生，只有一个变量不够。姓名、年龄、分数应该放在一起，这就是结构体适合做的事。

没有结构体会怎样？

messy.c
char name[20] = "Tom";
int age = 18;
double score = 92.5;

这些变量都在描述同一个学生，但散在外面。结构体可以把它们打包。

定义结构体

struct.c
struct Student {
    char name[20];
    int age;
    double score;
};

使用结构体

use_struct.c
struct Student s1 = {"Tom", 18, 92.5};

printf("姓名：%s\n", s1.name);
printf("年龄：%d\n", s1.age);
printf("分数：%.1f\n", s1.score);
. 的意思： s1.age 可以理解成“s1 这个学生对象里的 age 成员”。

结构体能用来描述什么？

学生 姓名、年龄、成绩。
汽车信号 车速、档位、制动状态。
游戏角色 血量、攻击力、坐标。
16

新手最常见错误：看懂报错，比不犯错更重要

写 C 一定会报错。别害怕，报错不是电脑骂你，是电脑给你线索。

常见错误地图

错误现象可能原因怎么想
少分号某条语句后面忘了 ;看报错行和上一行。
括号不匹配{}、()、[] 少了一个成对检查。
变量未声明使用变量前没声明，或变量名拼错检查名字是否一致。
格式占位符错%d、%f、%c 用错占位符要和变量类型匹配。
数组越界访问了不存在的位置长度 5 的数组最大下标是 4。
指针问题指针没初始化或地址错误先确认指针指向哪里。

调试三步法

1
缩小范围
不要在一大坨代码里猜。先确定问题大概出现在哪几行。
2
打印变量
用 printf 打印关键变量，看它是不是你想象的值。
3
相信实际结果
不要靠感觉。程序跑出来什么，就说明当前代码实际做了什么。
真正的入门标志： 不是你再也不写 bug，而是你看到 bug 不慌，知道从哪里开始查。

练习路线：每学一点，就写一点

示例：猜数字游戏

guess_number.c
#include <stdio.h>

int main(void) {
    int answer = 42;
    int guess;

    printf("猜一个 1 到 100 的数字：\n");

    while (1) {
        printf("请输入：");
        scanf("%d", &guess);

        if (guess > answer) {
            printf("太大了！\n");
        } else if (guess < answer) {
            printf("太小了！\n");
        } else {
            printf("猜对了！\n");
            break;
        }
    }

    return 0;
}

发布到 GitHub：让手机能打开，还能添加到桌面

这个页面本质上是静态网页，最适合放到 GitHub Pages。发布后你会得到一个网址，用手机打开后就可以添加到桌面，效果很像一个小 App。

推荐目标： 先做成 GitHub Pages 网页；然后手机浏览器打开；最后用“添加到主屏幕 / 添加到桌面”变成桌面入口。iPhone 用 Safari 最稳，安卓用 Chrome / Edge 最稳。

最简单文件结构

你只需要先准备这一个文件：

项目结构
c-language-guide/
└── index.html

把当前这个 HTML 保存成 index.html，上传到 GitHub 仓库根目录即可。

如果想更像 App：建议再加 2 个文件

这一步不是必须，但安卓 Chrome 更容易识别成可安装网页，离线缓存也更好。

推荐结构
c-language-guide/
├── index.html
├── manifest.json
└── sw.js

manifest.json 示例

manifest.json
{
  "name": "C 语言零基础入门",

更正式的版本可以再加 192x192 和 512x512 的 PNG 图标；没有图标也能先跑起来。

sw.js 示例

sw.js
const CACHE_NAME = 'c-guide-v1';
const FILES = ['./', './index.html', './manifest.json'];

self.addEventListener('install', (event) => {
  event.waitUntil(caches.open(CACHE_NAME).then((cache) => cache.addAll(FILES)));
});

self.addEventListener('fetch', (event) => {
  event.respondWith(
    caches.match(event.request).then((cached) => cached || fetch(event.request))
  );
});

如果你加了 manifest 和 sw.js，建议再做两处改动

第一，把 head 里的 manifest 那一行注释打开：

放到 head 里
<link rel="manifest" href="./manifest.json" />

第二，在底部 script 里加入 Service Worker 注册代码：

注册 Service Worker
if ('serviceWorker' in navigator) {
  window.addEventListener('load', () => {
    navigator.serviceWorker.register('./sw.js');
  });
}

GitHub Pages 托管步骤

新建仓库
登录 GitHub，点右上角 + → New repository。仓库名可以叫 c-language-guide。
上传文件
进入仓库后点 Add file → Upload files，上传 index.html。如果你做 PWA，也一起上传 manifest.json 和 sw.js。
提交保存
页面底部点 Commit changes。这一步相当于把网页文件保存到仓库里。
开启 Pages
进入仓库 Settings → Pages，Source 选择 Deploy from a branch，Branch 选择 main，目录选择 /(root)，然后保存。
等待生成网址
稍等一会儿，GitHub Pages 会给你一个类似 https://你的用户名.github.io/c-language-guide/ 的网址。

手机添加到桌面

iPhone / iPad 用 Safari 打开网址 → 点分享按钮 → 选择“添加到主屏幕”。
安卓 用 Chrome 或 Edge 打开网址 → 菜单 → “添加到主屏幕”或“安装应用”。
注意： GitHub Pages 发布的是公开网页，不要放隐私信息、公司内部代码、客户资料或任何敏感内容。
